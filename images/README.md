# 🖼️ MUFE 백신 — 카오스 이미지 풀

대장님이 만드시는 **100개 카오스 이미지**가 박힐 자리.

## 사용 방법

### 1. 이미지 파일 추가

이 폴더에 이미지 파일 박으세요:
```
chaos-001.jpg
chaos-002.jpg
chaos-003.jpg
...
chaos-100.jpg
```

또는 자유 이름:
```
보라꽃-우주.jpg
대문-격자.png
선과색-1.webp
```

### 2. challenge.js에 경로 추가

`netlify/functions/challenge.js` 파일 열기 →
`IMAGE_POOL` 배열에 경로 추가:

```javascript
const IMAGE_POOL = [
  '/images/chaos-001.jpg',
  '/images/chaos-002.jpg',
  '/images/chaos-003.jpg',
  // ... 100개
];
```

### 3. 배포

ZIP 다시 만들어서 Netlify에 던지면 끝.

## 권장 사양

- **크기**: 1200 × 800 픽셀 이상 (회전기 배경 풀스크린)
- **형식**: JPG (작은 용량) 또는 WebP (더 작음)
- **용량**: 한 장당 200KB 이하 권장
- **내용**: 
  - 보라꽃 + 우주 격자 + 카오스 수식
  - 신성기하학 (생명의 꽃, 메타트론의 큐브)
  - 선과 색의 조화 (PCT 핵심)
  - 어떤 디자인이든 — 위에 회전 단어가 어둡게 잘 보이게

## 자동 작동

- 인증 챌린지 발급 시마다 ***이 풀에서 무작위 1개 선택***
- 매번 다른 이미지 + 다른 단어 = ***시도자별 완전히 다른 화면***
- AI가 학습할 패턴 0 (*"화이트 노이즈"* 처리)
- 인간만 직관으로 단어 잡기 가능

## 확장

- 100개 → 1,000개 → 10,000개 → 무한
- 무료 API (Pollinations 등) 추가 가능
- 사용자 등급별 이미지 풀 분리 가능
- 시간대별·접속 위치별 다른 풀
